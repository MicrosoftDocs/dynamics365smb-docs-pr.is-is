# <a name="design-details-variance"></a><span data-ttu-id="3ff29-103">Hönnunarupplýsingar Frávik</span><span class="sxs-lookup"><span data-stu-id="3ff29-103">Design Details: Variance</span></span>
<span data-ttu-id="3ff29-104">Frávik er skilgreint sem mismunurinn á raunverulegum kostnaði og staðalkostnað, eins og lýst er í eftirfarandi formúlu.</span><span class="sxs-lookup"><span data-stu-id="3ff29-104">Variance is defined as the difference between the actual cost and the standard cost, as described in the following formula.</span></span>  

 <span data-ttu-id="3ff29-105">raunkostnaður – staðlaður kostnaður = frávik</span><span class="sxs-lookup"><span data-stu-id="3ff29-105">actual cost – standard cost = variance</span></span>  

 <span data-ttu-id="3ff29-106">Ef raunkostnaður breytist, til dæmis vegna þess að kostnaðarauki er bókaður á síðari dagsetningu, er frávikið uppfært samkvæmt því.</span><span class="sxs-lookup"><span data-stu-id="3ff29-106">If the actual cost changes, for example, because you post an item charge on a later date, then the variance is updated accordingly.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="3ff29-107">Endurmat hefur ekki áhrif á dreifni útreikning, því endurmat einungis breytir birgðavirði.</span><span class="sxs-lookup"><span data-stu-id="3ff29-107">Revaluation does not affect the variance calculation, because revaluation only changes the inventory value.</span></span>
