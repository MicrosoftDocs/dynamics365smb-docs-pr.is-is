# <a name="import-and-export-a-report-or-document-layout"></a><span data-ttu-id="c26e1-103">Flytja inn og út skýrsluútlit eða skjalaútlit</span><span class="sxs-lookup"><span data-stu-id="c26e1-103">Import and Export a Report or Document Layout</span></span>
<span data-ttu-id="c26e1-104">Hægt er að flytja inn og út fyrirliggjandi sérsniðin skýrsluútlit sem skrá í og úr staðsetningu á tölvu og neti.</span><span class="sxs-lookup"><span data-stu-id="c26e1-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="c26e1-105">Til dæmis er hægt að flytja út skýrsluútlit og senda síðan skrána til einhvers annars til breytinga.</span><span class="sxs-lookup"><span data-stu-id="c26e1-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="c26e1-106">Sá einstaklingur getur gert breytingarnar á útlitinu og skilað skránni til þín til að flytja aftur inn.</span><span class="sxs-lookup"><span data-stu-id="c26e1-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>  

> [!IMPORTANT]  
>  <span data-ttu-id="c26e1-107">Ekki er hægt að flytja innbyggð skýrsluútlit inn eða út.</span><span class="sxs-lookup"><span data-stu-id="c26e1-107">You cannot import or export built-in report layouts.</span></span>  

### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="c26e1-108">Til að flytja út skýrsluútlit í skrá</span><span class="sxs-lookup"><span data-stu-id="c26e1-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="c26e1-109">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Val á útliti skýrslu** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="c26e1-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="c26e1-110">Veldu röðina fyrir skýrsluna sem er með sérsniðna skýrsluútlitið sem þú vilt flytja út, og svo, á flipanum **Heim** í hópnum **Vinna** velurðu **Sérstillt útlit**.</span><span class="sxs-lookup"><span data-stu-id="c26e1-110">Select the row for the report that contains the custom report layout that you want to export, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  

3.  <span data-ttu-id="c26e1-111">Á síðunni **Skýrsluútlit** skal velja skýrsluútlit sem flytja á út í skrá og svo á flipanum **Heim** í hópnum **Vinna** velja **Flytja út útlit**.</span><span class="sxs-lookup"><span data-stu-id="c26e1-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then on the **Home** tab, in the **Process** group, choose **Export Layout**.</span></span>  

4.  <span data-ttu-id="c26e1-112">Í svarglugganum **Flytja út skrá** skal velja **Vista** og vista svo skrána í tölvu eða á neti.</span><span class="sxs-lookup"><span data-stu-id="c26e1-112">In the **Export File** dialog box, choose **Save**, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="c26e1-113">Til að flytja inn skýrsluútlit úr skrá</span><span class="sxs-lookup"><span data-stu-id="c26e1-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="c26e1-114">Tryggja þarf að viðeigandi skrá sem tilgreinir skýrsluútlitið sé til staðar í tölvunni eða á netinu.</span><span class="sxs-lookup"><span data-stu-id="c26e1-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="c26e1-115">Word-skýrsluútlitsskrá verður að vera með .docx-skráarendingu.</span><span class="sxs-lookup"><span data-stu-id="c26e1-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="c26e1-116">RDLC-skýrsluútlitsskrá verður að vera með .rdlc- eða .rdl-skráarendingu.</span><span class="sxs-lookup"><span data-stu-id="c26e1-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="c26e1-117">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Val á útliti skýrslu** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="c26e1-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="c26e1-118">Veldu röðina fyrir skýrsluna sem þú vilt flytja inn skýrsluútlitið, og svo, á flipanum **Heim** í hópnum **Vinna** veldu **Sérstillt útlit**.</span><span class="sxs-lookup"><span data-stu-id="c26e1-118">Select the row for the report to which you want to import the report layout, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  

4.  <span data-ttu-id="c26e1-119">Á síðunni **Skýrsluútlit** skal velja útlit sem flytja á inn í skrána og svo á flipanum **Heim** í hópnum **Vinna** velja **Flytja inn útlit**.</span><span class="sxs-lookup"><span data-stu-id="c26e1-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then on the **Home** tab, in the **Process** group, choose **Import Layout**.</span></span>  

5.  <span data-ttu-id="c26e1-120">Í svarglugganum **Flytja inn** skal velja skjalið sem skilgreinir skýrsluútlitið og svo velja **Opna**.</span><span class="sxs-lookup"><span data-stu-id="c26e1-120">In the **Import** dialog box, select the document that defines the report layout, and then choose **Open**.</span></span>  

 <span data-ttu-id="c26e1-121">Upprunalega sérsniðna skýrsluútlitinu er skipt út með innflutta skýrsluútlitinu.</span><span class="sxs-lookup"><span data-stu-id="c26e1-121">The original custom report layout is replaced with the imported report layout.</span></span>
