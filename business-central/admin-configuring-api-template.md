<span data-ttu-id="9cb81-114">Til að úthluta API sniðmáti verður þú að fara í gegnum eftirfarandi skref.</span><span class="sxs-lookup"><span data-stu-id="9cb81-114">To assign a template to an API, you must go through the following steps.</span></span>

<span data-ttu-id="9cb81-117">Ef fleiri en eitt sniðmát er valið fyrir API (síðukenni) eru sniðmátin notuð í þeirri röð sem er skilgreind í dálknum **Röðun**.</span><span class="sxs-lookup"><span data-stu-id="9cb81-117">If there is more than one template selected for an API (Page ID), the templates are applied in the order defined in the **Order** column.</span></span>   
<span data-ttu-id="9cb81-118">Þegar hvert sniðmát er notað eru reitargildum sem eru skilgreind í sniðmátinu aðeins beitt á reiti sem hafa ekki enn fengið gildi skilgreint, annaðhvort sérstaklega í API eða í sniðmáti sem notað var áður í röðinni.</span><span class="sxs-lookup"><span data-stu-id="9cb81-118">When each template is applied, field values defined in the template are only applied to fields that have not already had a value defined, either explicitly in the API, or in a previously applied template in the order.</span></span> 
3. <span data-ttu-id="9cb81-119">Veljið gildi fyrir **Síðukenni**.</span><span class="sxs-lookup"><span data-stu-id="9cb81-119">Select a **Page ID** value.</span></span>  
<span data-ttu-id="9cb81-120">Þetta er síðan fyrir API sem sniðmátið verður beitt á.</span><span class="sxs-lookup"><span data-stu-id="9cb81-120">This is the page for the API to which the template will be applied.</span></span> <span data-ttu-id="9cb81-121">Uppflettingin **Síðukenni** býður upp á lista af öllum API sem eru í boði í safninu.</span><span class="sxs-lookup"><span data-stu-id="9cb81-121">The **Page ID** lookup provides a list of all APIs available in the library.</span></span>
4. <span data-ttu-id="9cb81-122">Veljið gildi í reitnum **Sniðmátskóði**.</span><span class="sxs-lookup"><span data-stu-id="9cb81-122">Select a value in the **Template Code** field.</span></span>  
<span data-ttu-id="9cb81-123">Sniðmátskóðinn er kóðinn fyrir sniðmátið sem var skilgreint á síðunni **Grunnstillingarsniðmát**.</span><span class="sxs-lookup"><span data-stu-id="9cb81-123">The template code is the code for the template that was defined on the **Configuration Templates** page.</span></span> <span data-ttu-id="9cb81-124">Skilgreindum sniðmátsgildum er beitt á API.</span><span class="sxs-lookup"><span data-stu-id="9cb81-124">The template values defined are applied to the API.</span></span> 
5. <span data-ttu-id="9cb81-125">Í reitnum **Skilyrði** skal tilgreina hvaða sniðmátum eigi að beita.</span><span class="sxs-lookup"><span data-stu-id="9cb81-125">In the **Conditions** field, specify which template should be applied.</span></span>  
<span data-ttu-id="9cb81-126">Skilgreindu sniðmáti er beitt á nýja færslu sem er stofnuð í gegnum API, en eingöngu ef skilyrðin sem eru skilgreind í reitnum **Skilyrði** eru uppfyllt af gildunum sem þegar eru skilgreind fyrir nýja tilvikið af einingunni.</span><span class="sxs-lookup"><span data-stu-id="9cb81-126">The defined template is applied to a new record created through the API if, and only if, the conditions defined in the **Conditions** field are met by the values already defined for the new instance of the entity.</span></span>
