3. <span data-ttu-id="b8097-109">Veldur ritilinn þinn, PowerApps Studio fyrir Windows eða PowerApps Studio fyrir vefinn.</span><span class="sxs-lookup"><span data-stu-id="b8097-109">Choose your editor, PowerApps Studio for Windows or PowerApps Studio for Web.</span></span>

   <span data-ttu-id="b8097-110">PowerApps Studio fyrir Windows er skjáborðsforrit sem notað er til að stofna og birta PowerApps.</span><span class="sxs-lookup"><span data-stu-id="b8097-110">PowerApps Studio for Windows is a desktop application used to create and publish PowerApps.</span></span> <span data-ttu-id="b8097-111">PowerApps Studio fyrir vef er skjáborðsforrit sem notað er til að stofna og birta PowerApps.</span><span class="sxs-lookup"><span data-stu-id="b8097-111">The PowerApps Studio for Web is the online solution used to create and publish PowerApps.</span></span>

6. <span data-ttu-id="b8097-115">PowerApps birtir tengisíðu sem sendir áminningu um upplýsingarnar sem gefa þarf upp til að tengja notanda við [!INCLUDE[d365fin](includes/d365fin_md.md)]-gögnin sín.</span><span class="sxs-lookup"><span data-stu-id="b8097-115">PowerApps will display a connection page that prompts you for the information that is required to connect to your [!INCLUDE[d365fin](includes/d365fin_md.md)] data.</span></span> <span data-ttu-id="b8097-116">Til að tengjast verður að tilgreina OData-vefslóð, notandanafn, aðgangsorð og heiti fyrirtækis.</span><span class="sxs-lookup"><span data-stu-id="b8097-116">To connect, you must specify an OData URL, username, password, and company name.</span></span>

   <span data-ttu-id="b8097-117">Til að finna *OData vefslóð* getur notandi afritað OData V4 vefslóð fyrir hverja þeirra vefþjónusta sem taldar eru upp á gluggi **Vefþjónustur** í [!INCLUDE[d365fin](includes/d365fin_md.md)], svo sem `https://mycompany.businesscentral.dynamics.com:7048/MS/ODataV4/`.</span><span class="sxs-lookup"><span data-stu-id="b8097-117">For the *OData URL*, you can copy the OData V4 URL of any of the web services that are listed in the **Web Services** window in [!INCLUDE[d365fin](includes/d365fin_md.md)], such as `https://mycompany.businesscentral.dynamics.com:7048/MS/ODataV4/`.</span></span>  

   <span data-ttu-id="b8097-118">Fyrir *Fyrirtækisheiti* skal nota heitið sem birtist í reitnum **Heiti** í glugganum **Stofngögn** í [!INCLUDE[d365fin](includes/d365fin_md.md)].</span><span class="sxs-lookup"><span data-stu-id="b8097-118">For the *Company Name*, use the name that is shown in the **Name** field in the **Company Information** window in [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span> <span data-ttu-id="b8097-119">Ef [!INCLUDE[d365fin](includes/d365fin_md.md)] notanda inniheldur mörg fyrirtæki skal velja viðeigandi heiti fyrirtækis af listanum í glugganum **Fyrirtæki**.</span><span class="sxs-lookup"><span data-stu-id="b8097-119">If your [!INCLUDE[d365fin](includes/d365fin_md.md)] contains multiple companies, choose the relevant company name from the list in the **Companies** window.</span></span> <span data-ttu-id="b8097-120">Í báðum tilvikum skal gæta þess að heitið sem notandi tilgreinir í leiðsagnarforritinu fyrir PowerApps samsvari nákvæmlega textanum sem birtist í [!INCLUDE[d365fin](includes/d365fin_md.md)], svo sem `My Company`.</span><span class="sxs-lookup"><span data-stu-id="b8097-120">In both cases, make sure that the name that you specify in the PowerApps wizard matches exactly the text shown in [!INCLUDE[d365fin](includes/d365fin_md.md)], such as `My Company`.</span></span>

   <span data-ttu-id="b8097-121">Fyrir notandanafn og aðgangsorð skal nota nafnið og aðgangslykil vefþjónustu sem tilgreind eru í reikningnum í glugganum **Notendur** í [!INCLUDE[d365fin](includes/d365fin_md.md)].</span><span class="sxs-lookup"><span data-stu-id="b8097-121">For the username and password, use the name and web service access key that are specified for your account in the **Users** window in [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span> <span data-ttu-id="b8097-122">Til dæmis, notandanafnið þitt er *ADMIN*, og aðgangslykill vefþjónustu sem virkar sem aðgangsorð er *EgzeUFQ9Uv0o5O0lUMyqCzo1ueUW9yRF3SsLU=*.</span><span class="sxs-lookup"><span data-stu-id="b8097-122">For example, your username is *ADMIN*, and the web service access key that serves as your password is *EgzeUFQ9Uv0o5O0lUMyqCzo1ueUW9yRF3SsLU=*.</span></span>
7. <span data-ttu-id="b8097-123">Veldu hnappinn **Tenging** til að halda áfram.</span><span class="sxs-lookup"><span data-stu-id="b8097-123">Choose the **Connection** button to continue.</span></span> <span data-ttu-id="b8097-124">PowerApps birtir sjálfgefið gagnamengi fyrir [!INCLUDE[d365fin](includes/d365fin_md.md)].</span><span class="sxs-lookup"><span data-stu-id="b8097-124">PowerApps will display a default dataset for [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span> <span data-ttu-id="b8097-125">Veljið gagnamengið **Sjálfgefið**.</span><span class="sxs-lookup"><span data-stu-id="b8097-125">Choose the **Default** dataset.</span></span>

   <span data-ttu-id="b8097-128">Einnig er hægt að stofna nýja vefslóð vefþjónustu í [!INCLUDE[d365fin](includes/d365fin_md.md)] með því að nota aðgerðina **Stofna gagnamengi** á glugganum **Vefþjónustur** með því að nota Uppsetningu með hjálp fyrir **Setja upp skýrslugerð** eða með því að velja aðgerðina **Breyta í Excel** í hvaða lista sem er.</span><span class="sxs-lookup"><span data-stu-id="b8097-128">Alternatively, create a new web service URL in [!INCLUDE[d365fin](includes/d365fin_md.md)] by using the **Create Data Set** action in the **Web Services** window, using the **Set Up Reporting** Assisted Setup guide, or by choosing the **Edit in Excel** action in any lists.</span></span>
8. <span data-ttu-id="b8097-129">Veldu töfluna sem á að nota fyrir þitt PowerApp og veldu svo hnappinn **Tengja**.</span><span class="sxs-lookup"><span data-stu-id="b8097-129">Choose the table that you want to use for your PowerApp, and then choose the **Connect** button.</span></span>
9. <span data-ttu-id="b8097-130">Endurtaktu fyrri skref til að bæta fleiri [!INCLUDE[d365fin](includes/d365fin_md.md)]-gögnum við Power Bi-gagnalíkanið þitt.</span><span class="sxs-lookup"><span data-stu-id="b8097-130">Repeat the previous steps to add additional [!INCLUDE[d365fin](includes/d365fin_md.md)] data to your Power BI data model.</span></span>

   > [!NOTE]  
   >    <span data-ttu-id="b8097-131">Þegar notanda hefur tekist að tengjast [!INCLUDE[d365fin](includes/d365fin_md.md)] verður hann ekki beðinn aftur um OData-vefslóð, notandanafn eða aðgangsorð.</span><span class="sxs-lookup"><span data-stu-id="b8097-131">Once you have successfully connected to [!INCLUDE[d365fin](includes/d365fin_md.md)], you will not be prompted again for the OData URL, username, or password.</span></span>
