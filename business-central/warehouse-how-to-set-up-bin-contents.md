# <a name="create-bin-contents"></a><span data-ttu-id="9471b-103">Stofna innihald hólfs</span><span class="sxs-lookup"><span data-stu-id="9471b-103">Create Bin Contents</span></span>

## <a name="to-create-bin-content-manually"></a><span data-ttu-id="9471b-107">Stofna Innihald hólfs handvirkt</span><span class="sxs-lookup"><span data-stu-id="9471b-107">To create bin content manually</span></span>

1. <span data-ttu-id="9471b-108">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Staðsetningar** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="9471b-108">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Locations** , and then choose the related link.</span></span>  
2. <span data-ttu-id="9471b-109">Veljið birgðageymsluna þar sem á að setja upp hólfainnihald og veljið svo aðgerðina **Hólf** .</span><span class="sxs-lookup"><span data-stu-id="9471b-109">Select the location where you want to set up bin contents,  and then choose the **Bins** action.</span></span>  
3. <span data-ttu-id="9471b-110">Veljið hólfið þar sem á að setja upp innihald og veljið svo aðgerðina **Innihald** .</span><span class="sxs-lookup"><span data-stu-id="9471b-110">Select the bin where you want to set up contents, and then choose the **Contents** action.</span></span>  
4. <span data-ttu-id="9471b-111">Fyrir hverja vöru sem geyma á í hólfinu skal fylla út línu á síðunni **Innihald hólfs** með viðeigandi upplýsingum.</span><span class="sxs-lookup"><span data-stu-id="9471b-111">For each item that you want to store in the bin, fill in a line on the **Bin Contents** page with the appropriate information.</span></span> <span data-ttu-id="9471b-112">Nokkrir reitir eru þegar útfylltir með upplýsingum um hólfið.</span><span class="sxs-lookup"><span data-stu-id="9471b-112">Some of the fields are filled in already with information about the bin.</span></span>  
5. <span data-ttu-id="9471b-113">Fyrst er reiturinn **Vörunr.** fylltur út og, ef notaður er beinn frágangur og tínsla, eru aðrir reitir fylltir út, svo sem **Mælieiningarkóti** , **Hám.magn** og **Lágm.magn** .</span><span class="sxs-lookup"><span data-stu-id="9471b-113">First fill in the **Item No.** field, and then, if you are using directed put-away and pick, fill in the other fields such as the **Unit of Measure Code** , **Max. Qty.** , and **Min. Qty.** fields.</span></span>  

<span data-ttu-id="9471b-114">Veljið **Fast** ef þess þarf.</span><span class="sxs-lookup"><span data-stu-id="9471b-114">Select the **Fixed** field if necessary.</span></span> <span data-ttu-id="9471b-115">Ef hólfið á að vera sjálfgefið fyrir vöruna veljið reitinn **Sjálfgefið hólf** .</span><span class="sxs-lookup"><span data-stu-id="9471b-115">If the bin is to be used as the default bin for the item, select the **Default Bin** field.</span></span>  

<span data-ttu-id="9471b-116">Ef notaður er beinn frágangur og tínsla og réttar víddarupplýsingar hafa verið færðar inn á birgðaspjaldinu um mælieiningu hverrar vöru er hámarksmagnið sem fært er inn á síðuna **Innihald hólfs** borið samanvið raungetu hólfsins.</span><span class="sxs-lookup"><span data-stu-id="9471b-116">If you are using directed put-away and pick, and if you have entered the correct dimensional information on the item card about each item's units of measure, the maximum quantity that you enter on the **Bin Contents** page is verified against the physical capabilities of the bin.</span></span> <span data-ttu-id="9471b-117">Hámarks- og lágmarksmagn er síðan notað þegar áfylling hólfa er reiknuð og tillögur gerðar um frágang.</span><span class="sxs-lookup"><span data-stu-id="9471b-117">The minimum and maximum quantities are used when calculating bin replenishment and suggested put-aways.</span></span>  

<span data-ttu-id="9471b-118">Ef gátmerki er sett inn í reitinn **Fast** er varan fest við hólfið sem þýðir að  [!INCLUDE[d365fin](includes/d365fin_md.md)] reynir að setja þessa vöru í hólfið ef það er pláss fyrir hana og geyma færsluna sem festir vöruna við hólfið jafnvel þó magnið í hólfinu sé 0.</span><span class="sxs-lookup"><span data-stu-id="9471b-118">If you select the **Fixed** field, you are fixing the item to the bin, meaning that [!INCLUDE[d365fin](includes/d365fin_md.md)] will try to put this item in the bin if there is space for it, and it will preserve the record fixing the item to the bin even when the quantity in the bin is 0.</span></span> <span data-ttu-id="9471b-119">Kerfið getur sett aðrar vörur í hólfið þó að tiltekin vara hafi verið fest við hólfið.</span><span class="sxs-lookup"><span data-stu-id="9471b-119">Other items can be put into the bin, even though a particular item has been fixed to the bin.</span></span>  

> [!NOTE]  
> <span data-ttu-id="9471b-120">Hægt er að setja upp fleiri en eitt innihald hólfs samtímis á síðunni **Vinnublað f. stofnun hólfainnihalds** .</span><span class="sxs-lookup"><span data-stu-id="9471b-120">You can set up several bin contents at the same time on the **Bin Content Creation Worksheet** page.</span></span>  

## <a name="to-create-bin-content-with-a-worksheet"></a><span data-ttu-id="9471b-121">Stofna Innihald hólfs með vinnublaði</span><span class="sxs-lookup"><span data-stu-id="9471b-121">To create bin content with a worksheet</span></span>

<span data-ttu-id="9471b-122">Þegar lokið er við að stofna hólfin er hægt að stofna það innihald sem á að vera í hverju hólfi á stofnunarvinnublaði hólfainnihalds.</span><span class="sxs-lookup"><span data-stu-id="9471b-122">When you have created your bins, you can create the bin content that you want for each bin in the bin content creation worksheet.</span></span>

1. <span data-ttu-id="9471b-123">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Vinnublað f. stofnun hólfainnihalds** og veldu svo tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="9471b-123">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Bin Content Creation Worksheet** , and then choose the related link.</span></span>  
2. <span data-ttu-id="9471b-124">Á vinnublaðshausnum er smellt í reitinn **Heiti** og valið vinnublað birgðageymslunnar þar sem stofna á hólfainnihald.</span><span class="sxs-lookup"><span data-stu-id="9471b-124">On the worksheet header, in the **Name** field, select the worksheet of the location where you want to create bin contents.</span></span>

    <span data-ttu-id="9471b-126">Ef notaður er beinn frágangur og tínsla í birgðageymslunni er fyllt sjálfkrafa út reitina sem tilheyra hólfinu, svo sem **Tegund hólfs** , **Kóti vöruhúsaflokks** og **Hólfaflokkun** .</span><span class="sxs-lookup"><span data-stu-id="9471b-126">If you are using directed put-away and pick in this location, the fields relating to that particular bin, such as **Bin Type** , **Warehouse Class Code** , and **Bin Ranking** , will be filled in automatically.</span></span> <span data-ttu-id="9471b-127">Þetta eru upplýsingar sem þarf að hafa í huga þegar innihald hólfsins er skilgreint.</span><span class="sxs-lookup"><span data-stu-id="9471b-127">This is information that you need to consider as you define the bin content.</span></span>  
4. <span data-ttu-id="9471b-128">Varan sem úthluta á hólfinu er valin og reitir sem tengjast hólfainnihaldi fylltir út.</span><span class="sxs-lookup"><span data-stu-id="9471b-128">Select the item that you want to assign to the bin, and fill in the fields related to the bin content.</span></span> <span data-ttu-id="9471b-129">Ef notaður er beinn frágangur og tínsla og nota á aðgerðina **Reikna út áfyllingu hólfa** þarf að fylla út reitina **Hám.magn.** og **Lágm.magn** .</span><span class="sxs-lookup"><span data-stu-id="9471b-129">If you are using directed put-away and pick, and you want to use the **Calculate Bin Replenishment** function, fill in the **Max. Qty.** and **Min. Qty.** fields.</span></span>  

    <span data-ttu-id="9471b-130">Til að stilla þetta hólf sem æskilegt hólf fyrir vöruna jafnvel þó hólfamagnið sé **0** og öll önnur frágangsskilyrði séu jöfn skal velja reitinn **Fast** .</span><span class="sxs-lookup"><span data-stu-id="9471b-130">To set this bin as the preferred bin for the item even if the bin quantity is **0** and all other put-away criteria equal, select the **Fixed** field.</span></span>  
5. <span data-ttu-id="9471b-131">Skref 3 til 4 eru endurtekin fyrir hverja vöru sem á að úthluta hólfi.</span><span class="sxs-lookup"><span data-stu-id="9471b-131">Repeat steps 3 through 4 for each item you want to assign to a bin.</span></span>  
6. <span data-ttu-id="9471b-132">Velja aðgerðina **Prenta** til að forskoða og prenta hólfainnihaldið sem hefur verið fært inn á vinnublaðið.</span><span class="sxs-lookup"><span data-stu-id="9471b-132">Choose the **Print** action to preview and print the bin content you have entered in the worksheet.</span></span> <span data-ttu-id="9471b-133">Haldið er áfram að yfirfara innihald hólfa þar til notandinn er ánægður.</span><span class="sxs-lookup"><span data-stu-id="9471b-133">Continue to revise the bin content until you are satisfied.</span></span>  
7. <span data-ttu-id="9471b-134">Þegar þú ert tilbúin(n) er valið **Stofna innihald hólfa** aðgerð.</span><span class="sxs-lookup"><span data-stu-id="9471b-134">When you are ready, choose the **Create Bin Content** action.</span></span>  

<span data-ttu-id="9471b-135">Á þessu vinnublaði er hægt að vinna með nokkrar hólfainnihaldslínur fyrir nokkur hólf og fá þannig gott yfirlit yfir hvað er sett í hin ýmsu hólf á tilteknu svæði, gangi eða rekka.</span><span class="sxs-lookup"><span data-stu-id="9471b-135">In this worksheet, you can work with a number of bin content lines for a number of bins and thereby obtain a good overview of what you are putting into various bins in a given zone, aisle, or rack.</span></span>
