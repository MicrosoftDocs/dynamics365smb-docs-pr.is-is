## <a name="to-set-up-cost-allocations"></a><span data-ttu-id="b8cdd-109">Til að setja upp kostnaðarúthlutanir</span><span class="sxs-lookup"><span data-stu-id="b8cdd-109">To set up cost allocations</span></span>  
1.  <span data-ttu-id="b8cdd-110">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Kostnaðarúthlutun** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-110">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Cost Allocation**, and then chose the related link.</span></span>  
2.  <span data-ttu-id="b8cdd-111">Á síðunni **Kostnaðarúthlutun** skal velja reitinn **Breyta**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-111">On the **Cost Allocation** page, choose the **Edit** action.</span></span>  
3.  <span data-ttu-id="b8cdd-112">Færið inn kenni fyrir úthlutunaruppsprettu í reitnum **Kenni**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-112">Enter an ID for the allocation source in the **ID** field.</span></span>  
4.  <span data-ttu-id="b8cdd-113">Skilgreina stig sem tölu á bilinu 1 til 99 í reitnum **Stig**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-113">Define a level as a number between 1 and 99 in the **Level** field.</span></span> <span data-ttu-id="b8cdd-114">Úthlutunarbókunin mun fylgja röð stiganna.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-114">The allocation posting will follow the order of the levels.</span></span>  
5.  <span data-ttu-id="b8cdd-115">Færa inn kostnaðartegund til að skilgreina hvaða kostnaðartegundum verður úthlutað í reitinn **Svið kostnaðartegundar**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-115">Enter a cost type to define which cost types will be allocated in the **Cost Type Range** field.</span></span> <span data-ttu-id="b8cdd-116">Ef öllum kostnaði kostnaðartegundar hefur verið úthlutað er ekkert svið skilgreint.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-116">If all costs for a cost type are allocated, no range is defined.</span></span>  
6.  <span data-ttu-id="b8cdd-117">Færa inn kostnaðarstað ásamt kostnaði sem á að úthluta í reitinn **Kostnaðarstaðarkóði**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-117">Enter a cost center together with costs to be allocated in the **Cost Center Code** field.</span></span>  
7.  <span data-ttu-id="b8cdd-118">Færa inn kostnaðarhlut ásamt kostnaði sem á að úthluta í reitinn **Kostnaðarhlutakóði**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-118">Enter a cost object together with costs to be allocated in the **Cost Object Code** field.</span></span> <span data-ttu-id="b8cdd-119">Reiturinn er oftast áfram auður vegna þess að kostnaðarhlutir eru sjaldnast úthlutaðir á aðra kostnaðarhluti.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-119">Most often, this field stays empty, because cost objects are rarely allocated to other cost objects.</span></span>  
8.  <span data-ttu-id="b8cdd-120">Færa inn kostnaðargerð í reitinn **Kreditfært í kostnaðartegund**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-120">Enter a cost type in the **Credit to Cost Type** field.</span></span> <span data-ttu-id="b8cdd-121">Kostnaður sem er úthlutað verður kreditfærður í upprunakostnaðartegund.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-121">The costs that are allocated will be credited to the source cost type.</span></span> <span data-ttu-id="b8cdd-122">Kreditbókunin verður bókuð á þá kostnaðartegund sem uppgefin er hér.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-122">The credit posting will be posted to the cost type given here.</span></span>  
9. <span data-ttu-id="b8cdd-123">Á flýtiflipanum **Línur** skal skilgreina úthlutunarmörk.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-123">On the **Lines** FastTab, define the allocation targets.</span></span> <span data-ttu-id="b8cdd-124">Í fyrstu línunni skal færa inn kostnaðartegund í reitinn **Markkostnaðartegund**.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-124">On the first line, enter a cost type in the **Target Cost Type** field.</span></span> <span data-ttu-id="b8cdd-125">Skilgreinir hvaða kostnaðartegund úthlutunin er skuldfærð á.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-125">It defines which cost type the allocation is debited to.</span></span>  
10. <span data-ttu-id="b8cdd-126">Í fyrstu línunni skal færa inn fyrsta úthlutunar markið í reitinn **Markkostnaðarstaður** eða reitinn **Markkostnaðarhlutur** .</span><span class="sxs-lookup"><span data-stu-id="b8cdd-126">On the first line, enter the first allocation target in the **Target Cost Center** field or **Target Cost Object** the field.</span></span> <span data-ttu-id="b8cdd-127">Þessir tveir reitir skilgreina hvaða kostnaðarstað eða kostnaðarhlut úthlutunin er skuldfærð á.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-127">These two fields define which cost center or cost object the allocation is debited to.</span></span> <span data-ttu-id="b8cdd-128">Aðeins er hægt að færa inn í gildi annars hvors þessara reita, en ekki bæði.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-128">You can only fill in one of these fields, but not both.</span></span>  
11. <span data-ttu-id="b8cdd-129">Endurtaka skal sömu skref í annarri línu til að setja upp fleiri úthlutunarmörk.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-129">Repeat the same steps on the second line to set up additional allocation targets.</span></span>  
12. <span data-ttu-id="b8cdd-130">Þegar búið er að setja upp úthlutunarmarkmið og -uppruna, skal velja aðgerðina **Reikna úthlutunarlykil** til að reikna samtölu hlutdeildargilda.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-130">After you have set up the allocation target and sources, choose the **Calculate Allocation Key** action to calculate the total share values.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="b8cdd-131">Veljið gátreitinn **Útilokað** til að gera úthlutunaruppsetningu óvirka.</span><span class="sxs-lookup"><span data-stu-id="b8cdd-131">Select the **Blocked** check box to deactivate the allocation setup.</span></span>
