# <a name="set-up-item-units-of-measure"></a><span data-ttu-id="49f82-103">Setja upp mælieiningu vara</span><span class="sxs-lookup"><span data-stu-id="49f82-103">Set Up Item Units of Measure</span></span>
<span data-ttu-id="49f82-104">Hægt er að setja upp margar mælieiningar fyrir vöru þannig að hægt sé að úthluta mælieiningum til vörunnar í eftirfarandi tilgangi:</span><span class="sxs-lookup"><span data-stu-id="49f82-104">You can set up multiple units of measure for an item so that you can assign units of measure to the item for the following purposes:</span></span>

- <span data-ttu-id="49f82-105">Úthluta grunnmælieiningu á birgðaspjaldi vörunnar til skilgreiningar á því hvernig hún er geymd í birgðum og þjónar sem umbreytigrundvöllur fyrir aðrar mælieiningar.</span><span class="sxs-lookup"><span data-stu-id="49f82-105">Assign a base unit of measure on the item’s item card to define how it is stored in inventory and to serve as the conversion basis for alternate units of measure.</span></span>
- <span data-ttu-id="49f82-106">Úthluta öðrum mælieiningum til innkaupa, framleiðslu eða söluskjala til skilgreiningar á hversu margar einingar af grunnmælieiningunum eru afgreiddar á sama tíma í ferlinu.</span><span class="sxs-lookup"><span data-stu-id="49f82-106">Assign alternate units of measure to purchase, production, or sales documents to specify how many units of the base unit of measure you handle at a time in those processes.</span></span> <span data-ttu-id="49f82-107">Til dæmis er varan hugsanlega keypt á brettum en er notuð í stykkjatali við framleiðslu.</span><span class="sxs-lookup"><span data-stu-id="49f82-107">For example, you may buy the item on pallets and only use single pieces in your production.</span></span>

<span data-ttu-id="49f82-108">Ef vara er sett á lager eftir einni mælieiningu en framleidd eftir annarri er framleiðslupöntun stofnuð sem notar mælieiningu framleiðslukeyrslu til að reikna út rétt magn íhluta meðan á keyrslunni **Endurnýjun framleiðslupöntunar** stendur.</span><span class="sxs-lookup"><span data-stu-id="49f82-108">If an item is stocked in one unit of measure but produced in another, a production order is created that uses a manufacturing batch unit of measure to calculate the correct quantity of the components during the **Refresh Production Order** batch job.</span></span> <span data-ttu-id="49f82-109">Dæmi um útreikning með mælieiningu framleiðslukeyrslu er þegar framleiddur hlutur er merktur á lager í stykkjum en framleiddur í tonnum.</span><span class="sxs-lookup"><span data-stu-id="49f82-109">An example of a manufacturing batch unit of measure calculation is when a manufactured item is stocked in pieces but produced in tons.</span></span> <span data-ttu-id="49f82-110">Frekari upplýsingar eru í [Vinna með mælieiningu framleiðslukeyrslu](production-how-to-use-the-manufacturing-batch-unit-of-measure.md).</span><span class="sxs-lookup"><span data-stu-id="49f82-110">For more information, see [Work with Manufacturing Batch Units of Measure](production-how-to-use-the-manufacturing-batch-unit-of-measure.md).</span></span>

## <a name="to-set-up-a-unit-of-measure"></a><span data-ttu-id="49f82-111">Til að setja upp mælieiningu</span><span class="sxs-lookup"><span data-stu-id="49f82-111">To set up a unit of measure</span></span>
1. <span data-ttu-id="49f82-112">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Vörur** og veldu síðan tengda hlekkinn.</span><span class="sxs-lookup"><span data-stu-id="49f82-112">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="49f82-113">Opna spjald vörunnar sem á að setja upp aðra mælieiningu fyrir.</span><span class="sxs-lookup"><span data-stu-id="49f82-113">Open the card of the item for which you want to set up alternate units of measure.</span></span>
3. <span data-ttu-id="49f82-114">Veldu **Mælieining**.</span><span class="sxs-lookup"><span data-stu-id="49f82-114">Choose the **Units of Measure** action.</span></span> <span data-ttu-id="49f82-115">Síðan **Mælieiningar vöru** opnast.</span><span class="sxs-lookup"><span data-stu-id="49f82-115">The **Item Units of Measure** page opens.</span></span>
4. <span data-ttu-id="49f82-116">Ef reiturinn **Grunneining mælingareits** á birgðaspjaldinu er fylltur út, hefur sú mælieining þegar verið sett upp.</span><span class="sxs-lookup"><span data-stu-id="49f82-116">If the **Base Unit of Measure** field on the item card is filled, then that unit of measure is already set up.</span></span>
5. <span data-ttu-id="49f82-117">Valið er **Nýtt** aðgerð.</span><span class="sxs-lookup"><span data-stu-id="49f82-117">Choose the **New** action.</span></span> <span data-ttu-id="49f82-118">Ný auð lína er sett inn.</span><span class="sxs-lookup"><span data-stu-id="49f82-118">A new empty line is inserted.</span></span>
6. <span data-ttu-id="49f82-119">Í reitnum **Kóði**, setjið inn heiti mælieiningarinnar.</span><span class="sxs-lookup"><span data-stu-id="49f82-119">In the **Code** field, enter the name of the unit of measure.</span></span> <span data-ttu-id="49f82-120">Að öðrum kosti skal velja reitinn til að velja úr mælieiningarkóðum í gagnagrunninum.</span><span class="sxs-lookup"><span data-stu-id="49f82-120">Alternatively, choose the field to select from the unit of measure codes that are in the database.</span></span>
7. <span data-ttu-id="49f82-121">Í reitnum **Magn á mælieiningu**, tilgreinið hversu margar grunnmælieiningar varan inniheldur.</span><span class="sxs-lookup"><span data-stu-id="49f82-121">In the **Qty. per Unit of Measure** field, enter how many units of the base unit of measure the new unit of measure contains.</span></span>
8. <span data-ttu-id="49f82-122">Eindurtakið skref 5 til 7 til að setja upp allar aðrar mælieiningar sem nota á í mismunandi ferlum fyrir þessa vöru.</span><span class="sxs-lookup"><span data-stu-id="49f82-122">Repeat steps 5 through 7 to set up all the alternate units of measure that you want to use in different processes for this item.</span></span>

<span data-ttu-id="49f82-123">Nú er hægt að nota aðrar mælieiningar í innkaupa-, framleiðslu- og söluskjölum.</span><span class="sxs-lookup"><span data-stu-id="49f82-123">You can now use the alternate units of measure on purchase, production, and sales documents.</span></span> <span data-ttu-id="49f82-124">Frekari upplýsingar er að finna í Hvernig á að færa inn sjálfgefna mælieiningakóða fyrir innkaupafærslur og sölufærslur eða Nota mælieiningakeyrslu framleiðslu.</span><span class="sxs-lookup"><span data-stu-id="49f82-124">For more information, see Enter Default Units of Measure Codes for Purchase Transactions and Sales Transactions or Use the Manufacturing Batch Unit of Measure.</span></span>

## <a name="to-set-up-unit-of-measure-translations"></a><span data-ttu-id="49f82-125">Setja upp mælieiningatexta</span><span class="sxs-lookup"><span data-stu-id="49f82-125">To set up unit of measure translations</span></span>
<span data-ttu-id="49f82-126">Þegar selt er til erlendra viðskiptavina er hægt að tilgreina mælieininguna á viðeigandi tungumáli.</span><span class="sxs-lookup"><span data-stu-id="49f82-126">When you sell items to foreign customers, you may want to specify the unit of measure in the customer’s language.</span></span> <span data-ttu-id="49f82-127">Þetta er hægt að gera þegar búið er að setja upp nauðsynlega mælieiningatexta.</span><span class="sxs-lookup"><span data-stu-id="49f82-127">You can do this after you have set up the necessary unit of measure translations.</span></span>

1. <span data-ttu-id="49f82-128">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Mælieiningar** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="49f82-128">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Units of Measure**, and then choose the related link.</span></span>
2. <span data-ttu-id="49f82-129">Velja þarf tungumálakóðann sem setja á upp þýðingar fyrir og svo aðgerðina **Þýðingar**.</span><span class="sxs-lookup"><span data-stu-id="49f82-129">Select the code for which you want to set up translations, and then choose the **Translations** action.</span></span>
3. <span data-ttu-id="49f82-130">Í reitnum **Tungumálskóti** er felliörin valin til að skoða lista yfir tiltæka tungumálakóta.</span><span class="sxs-lookup"><span data-stu-id="49f82-130">In the **Language Code** field, select the drop-down arrow to see a list of available language codes.</span></span> <span data-ttu-id="49f82-131">Valinn er sá tungumálskóti sem setja á inn þýðingu fyrir og síðan er smellt á Í lagi til að afrita kótann í reitinn.</span><span class="sxs-lookup"><span data-stu-id="49f82-131">Select the language code for which you want to enter a translation, and then choose the OK button to copy the code to the field.</span></span>
4. <span data-ttu-id="49f82-132">Í reitinn **Lýsing** er færður inn viðeigandi texti.</span><span class="sxs-lookup"><span data-stu-id="49f82-132">In the **Description** field, enter the appropriate text.</span></span>
5. <span data-ttu-id="49f82-133">Skref 2 til 4 eru endurtekin fyrir þá mælieiningarkóða og tungumál sem setja á inn þýðingar á.</span><span class="sxs-lookup"><span data-stu-id="49f82-133">Repeat steps 2 through 4 for the unit of measure codes and the languages for which you want to enter translations.</span></span>

2. <span data-ttu-id="49f82-138">Opna viðeigandi birgðaspjaldi þar sem á að tilgreina sjálfgefinn mælieiningarkóða fyrir sölu- eða innkaupaeiningu.</span><span class="sxs-lookup"><span data-stu-id="49f82-138">Open the relevant item card for which you want to specify a default sales or purchase unit of measure code.</span></span>
3. <span data-ttu-id="49f82-139">Fyrir sölu, á flýtiflipanum **Reikningsfærsla**, í reitnum **Sölumælieining** er síðan **Mælieiningar vöru** opnuð.</span><span class="sxs-lookup"><span data-stu-id="49f82-139">For sales, on the **Invoicing** FastTab, in the **Sales Unit of Measure** field, open the **Item Units of Measure** page.</span></span>
