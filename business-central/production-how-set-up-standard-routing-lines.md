## <a name="to-set-up-a-standard-task"></a><span data-ttu-id="3b69a-106">Setja upp staðlaðan verkhluta</span><span class="sxs-lookup"><span data-stu-id="3b69a-106">To set up a standard task</span></span>

1. <span data-ttu-id="3b69a-107">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Staðalverkhlutar** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="3b69a-107">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Standard Tasks**, and then choose the related link.</span></span>
2. <span data-ttu-id="3b69a-108">Valið er **Nýtt** aðgerð.</span><span class="sxs-lookup"><span data-stu-id="3b69a-108">Choose the **New** action.</span></span>
3. <span data-ttu-id="3b69a-109">Fyllið inn í reitina eftir þörfum.</span><span class="sxs-lookup"><span data-stu-id="3b69a-109">Fill in the fields as necessary.</span></span> [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
4. <span data-ttu-id="3b69a-110">Velja **Verkfæri**, **Starfsmaður**, **Lýsing** eða **Verkfæri** aðgerðina og svo viðeigandi færslu á síðunni sem opnast.</span><span class="sxs-lookup"><span data-stu-id="3b69a-110">Choose the **Tools**, **Personnel**, **Description**, or **Tools** action, and then, on the page that opens, add the relevant entry.</span></span>

## <a name="to-add-a-standard-task-to-an-operation"></a><span data-ttu-id="3b69a-111">Bæta stöðluðum verkhluta við aðgerð</span><span class="sxs-lookup"><span data-stu-id="3b69a-111">To add a standard task to an operation</span></span>

1. <span data-ttu-id="3b69a-112">Veldu ![Ljósaperuna sem opnar eiginleika Viðmótsleitar](media/ui-search/search_small.png "Segðu mér hvað þú vilt gera") táknið, sláðu inn **Leið** og veldu síðan tengda tengilinn.</span><span class="sxs-lookup"><span data-stu-id="3b69a-112">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Routings**, and then choose the related link.</span></span>
2. <span data-ttu-id="3b69a-113">Viðeigandi leið er opnuð.</span><span class="sxs-lookup"><span data-stu-id="3b69a-113">Open the relevant routing.</span></span>
3. <span data-ttu-id="3b69a-114">Velja viðeigandi leiðarlínu, velja aðgerðina **Aðgerðir** og veljið síðan **Verkfæri**, **Starfsmaður**, **Lýsing** eða **Verkfæri** aðgerðina.</span><span class="sxs-lookup"><span data-stu-id="3b69a-114">Select a relevant routing line, choose the **Operations** action, and then choose the **Tools**, **Personnel**, **Description**, or **Tools** action.</span></span>

<span data-ttu-id="3b69a-115">Þegar leiðarblað er prentað er hægt að velja að taka með einingarnar úr staðlaða verkinu sem bætt var við leiðarlínurnar.</span><span class="sxs-lookup"><span data-stu-id="3b69a-115">When you print the routing sheeting, you can choose to include the elements from the standard task that you added to the routing lines.</span></span>
