# <a name="sell-items-assembled-to-order"></a><span data-ttu-id="4f14e-104">Selja hluti sem eru settir saman í pöntun</span><span class="sxs-lookup"><span data-stu-id="4f14e-104">Sell Items Assembled to Order</span></span>
<span data-ttu-id="4f14e-105">Ef reiturinn **Samsetningarstefna** á birgðaspjaldi samsetningarvörunnar er **Samsetning til pöntunar** er ekki gert ráð fyrir því að varan sé í birgðum og þá þarf að setja hana saman fyrir þessa sölupöntun.</span><span class="sxs-lookup"><span data-stu-id="4f14e-105">If the **Assembly Policy** field on the item card of an assembly item is **Assemble-to-Order**, then the item is not expected to be in inventory, and it must be assembled specifically to a sales order.</span></span> <span data-ttu-id="4f14e-106">Þegar varan er sett inn í sölupöntunarlínu er samsetningarpöntun síðan búin til sjálfkrafa og tengd við sölupöntunina.</span><span class="sxs-lookup"><span data-stu-id="4f14e-106">When you enter the item on a sales order line, then an assembly order is automatically created and linked to the sales order.</span></span>

2.  <span data-ttu-id="4f14e-114">Stofnið sölupöntun.</span><span class="sxs-lookup"><span data-stu-id="4f14e-114">Create a sales order.</span></span> <span data-ttu-id="4f14e-115">Frekari upplýsingar eru í [Selja vörur](sales-how-sell-products.md).</span><span class="sxs-lookup"><span data-stu-id="4f14e-115">For more information, see [Sell Products](sales-how-sell-products.md).</span></span>  
3.  <span data-ttu-id="4f14e-116">Í reitnum **númer**</span><span class="sxs-lookup"><span data-stu-id="4f14e-116">In the **No.**</span></span> <span data-ttu-id="4f14e-117">færið inn vöru sem stillt eru til að setja saman í pöntun.</span><span class="sxs-lookup"><span data-stu-id="4f14e-117">field, enter an item that is set up to be assembled to order.</span></span>

5.  <span data-ttu-id="4f14e-120">Í reitnum **Magn** er fært inn hversu margar einingar á að selja.</span><span class="sxs-lookup"><span data-stu-id="4f14e-120">In the **Quantity** field, enter how many units to sell.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="4f14e-121">Ef einn eða fleiri íhlutir umbeðins magns samsetningarvöru eru ekki tiltækir birtist nákvæm ráðstöfunarviðvörunarsíða.</span><span class="sxs-lookup"><span data-stu-id="4f14e-121">If one or more components of the requested assembly item quantity are not available, then a detailed availability warning page opens.</span></span> <span data-ttu-id="4f14e-122">Nánari upplýsingar, sjá samsetning í boði.</span><span class="sxs-lookup"><span data-stu-id="4f14e-122">For more information, see Assembly Availability.</span></span>  

    <span data-ttu-id="4f14e-123">Samsetningarpöntun er nú stofnuð sjálfkrafa og tengd við sölupöntunarlínuna.</span><span class="sxs-lookup"><span data-stu-id="4f14e-123">An assembly order is now automatically created and linked to the sales order line.</span></span> <span data-ttu-id="4f14e-124">Gjalddagi þessarar samsetningarpöntunar er samstilltur við afhendingardagsetningu sölupöntunarlínunnar.</span><span class="sxs-lookup"><span data-stu-id="4f14e-124">The due date of this assembly order is synchronized with the shipment date of the sales order line.</span></span>
