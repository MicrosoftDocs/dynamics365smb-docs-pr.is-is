# <a name="set-up-a-colored-indicator-on-cues-for-the-company-or-individual-users"></a><span data-ttu-id="bc860-103">Setja upp litaðan vísi á bunka fyrir fyrirtækið eða einstaka notendur</span><span class="sxs-lookup"><span data-stu-id="bc860-103">Set Up a Colored Indicator on Cues for the Company or Individual Users</span></span>
<span data-ttu-id="bc860-104">Sem stjórnandi geturðu sett upp bendingar sem birtast í Mitt hlutverk notenda þannig að það innihaldi vísi sem breytir um lit eftir gagnagildi í bendingum.</span><span class="sxs-lookup"><span data-stu-id="bc860-104">As an administrator, you can set up Cues that appear on the users' Role Centers to include an indicator that changes color based on the data values in the Cues.</span></span>  
  
<span data-ttu-id="bc860-105">Vísirinn birtist sem stika efst í ramma bendingarinnar.</span><span class="sxs-lookup"><span data-stu-id="bc860-105">The indicator appears as a colored bar along the top border of the Cue tile.</span></span> <span data-ttu-id="bc860-106">Veitir sjónrænt merki um virknistöðu bendingarinnar sem getur táknað hagstæð eða óhagstæð skilyrði sem kalla á viðbrögð frá notanda.</span><span class="sxs-lookup"><span data-stu-id="bc860-106">It provides a visual signal of the status of the Cue's activity, which can indicate favorable or unfavorable conditions to prompt the user to take action.</span></span> <span data-ttu-id="bc860-107">Ef bunki birtir til dæmis viðvarandi sölureikninga er hægt að stilla vísinn á að vera grænn (jákvætt) þegar heildarfjöldi viðvarandi sölureikninga er undir 10 og rauður (óæskilegt) þegar samtalan er yfir 20.</span><span class="sxs-lookup"><span data-stu-id="bc860-107">For example, if a Cue displays ongoing sales invoices, you can set up the indicator to appear green (favorable) when total number of ongoing sales invoices is below 10, and appears red (unfavorable) when the total is greater than 20.</span></span>  
  
<span data-ttu-id="bc860-108">Á síðunni **uppsetning bunka** seturðu setja upp vísa fyrir allar bendingar sem eru tiltækar í gagnagrunni fyrirtækisins.</span><span class="sxs-lookup"><span data-stu-id="bc860-108">From the **Cue Setup** page, you set up indicators for all the Cues that are available in the company database.</span></span> <span data-ttu-id="bc860-109">Hægt er að setja upp vísa sem eru notaðir fyrir alla notendur í fyrirtækinu eða einstakan starfsmann.</span><span class="sxs-lookup"><span data-stu-id="bc860-109">You can set up the indicators to apply to all users in the company or an individual user only.</span></span> <span data-ttu-id="bc860-110">Vísisstillingarnar á síðunni **Uppsetning bunka** virka sem sjálfgefnar vísisstillingar.</span><span class="sxs-lookup"><span data-stu-id="bc860-110">The indicator settings on the **Cue Setup** page act as the default indicator settings.</span></span> <span data-ttu-id="bc860-111">Ef síðan **Uppsetning bendingar notandi** er gerð tiltæk fyrir notendur geta þeir sérsniðið vísastillingarnar sem eru skilgreindar á síðunni **Uppsetning bendingar**.</span><span class="sxs-lookup"><span data-stu-id="bc860-111">If the **Cue Setup End User** page is made available to users, then they can personalize the indicators settings that you define on the **Cue Setup** page.</span></span>  
  
<span data-ttu-id="bc860-112">Til að setja upp vísi tilgreinirðu allt að tvö þröskuldsgildi sem tilgreina þrjú svið gagnagilda (lágt, miðlungs og hátt) sem hægt er að nota annan lit (eða stíl) við.</span><span class="sxs-lookup"><span data-stu-id="bc860-112">To set up the indicator, you specify up to two threshold values that define three ranges of data values (low, middle, and high) to which you can apply a different color (or style).</span></span>
