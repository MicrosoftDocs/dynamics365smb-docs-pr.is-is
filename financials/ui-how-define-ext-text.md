<span data-ttu-id="641e5-104">Hægt er að lengja staðaltexta fyrir vörur með því að bæta við línum, og hægt er að setja upp skilyrði fyrir notkun á viðbótarlínunum.</span><span class="sxs-lookup"><span data-stu-id="641e5-104">You can extend a standard text for items by adding extra lines, and you can set up conditions for use of the extra lines.</span></span> <span data-ttu-id="641e5-105">Þetta er hægt að gera frá vöruspjöldum.</span><span class="sxs-lookup"><span data-stu-id="641e5-105">You do this from item cards.</span></span>

## <a name="to-define-extended-text-for-an-item-description"></a><span data-ttu-id="641e5-106">Til að skilgreina lengdan texta fyrir vörulýsingu</span><span class="sxs-lookup"><span data-stu-id="641e5-106">To define extended text for an item description</span></span>
1. <span data-ttu-id="641e5-107">Opnið spjaldið fyrir vöru sem bæta á við texta fyrir og veljið aðgerðina **Lengdur texti**.</span><span class="sxs-lookup"><span data-stu-id="641e5-107">Open the card for an item that you want to add extended text to, and then choose the **Extended Text** action.</span></span>
2. <span data-ttu-id="641e5-108">Í reitinn **Kóti** er færður inn kótinn og í reitinn **Lýsing** er færður inn texti til útskýringar.</span><span class="sxs-lookup"><span data-stu-id="641e5-108">In the **Code** field, enter the code, and in the **Description** field, enter the desired text.</span></span>
3. <span data-ttu-id="641e5-109">Veljið **Lengdur texti**.</span><span class="sxs-lookup"><span data-stu-id="641e5-109">Choose **Extended Texts**.</span></span>
4. <span data-ttu-id="641e5-110">Fylla inn í línurnar í glugganum **Lengdur texti** með viðbótartexta.</span><span class="sxs-lookup"><span data-stu-id="641e5-110">Fill in the lines in the **Extended Text** window with the additional text.</span></span>
5. <span data-ttu-id="641e5-111">Fyllið inn í reitinn **Tungumálakóði** eða reitinn **Allir tungumálakóðar** ef notaðir eru tungumálakóðar.</span><span class="sxs-lookup"><span data-stu-id="641e5-111">Fill in the **Language Code** field or the **All Language Codes** field if you use language codes.</span></span>
6. <span data-ttu-id="641e5-112">Reitirnir **Upphafsdagsetning** og **Lokadagsetning** eru fylltir út ef afmarka á tímabil fyrir lengdan texta.</span><span class="sxs-lookup"><span data-stu-id="641e5-112">Fill in the **Starting Date** and **Ending Date** fields if you want to limit the dates on which the extended text is used.</span></span>

8. <span data-ttu-id="641e5-114">Glugganum er lokað.</span><span class="sxs-lookup"><span data-stu-id="641e5-114">Close the window.</span></span>
