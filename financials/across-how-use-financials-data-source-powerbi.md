2. <span data-ttu-id="becb9-109">Í reitnum **Sækja gögn** skal velja **Netþjónustu**, velja **Dynamics 365 for Financials**, og velja svo hnappinn **Tengja**.</span><span class="sxs-lookup"><span data-stu-id="becb9-109">In the **Get Data** window, choose **Online Services**, choose **Dynamics 365 for Financials**, and then choose the **Connect** button.</span></span>

   <span data-ttu-id="becb9-110">Power BI birtir leiðsagnarforrit sem leiðbeinir þér gegnum tengingarferlið.</span><span class="sxs-lookup"><span data-stu-id="becb9-110">Power BI displays a wizard that will guide you though the connection process.</span></span> <span data-ttu-id="becb9-111">Fyrsta skref er notanda er að færa inn OData-vefslóð og heiti fyrirtækis sem tengist [!INCLUDE[d365fin](includes/d365fin_md.md)]-reikningnum hans.</span><span class="sxs-lookup"><span data-stu-id="becb9-111">The first step will be to enter an OData URL and the company name that is associated with your [!INCLUDE[d365fin](includes/d365fin_md.md)] account.</span></span>  

   <span data-ttu-id="becb9-112">Til að finna *OData vefslóð* getur notandi afritað OData V4 vefslóð fyrir hverja þeirra vefþjónusta sem taldar eru upp á síðunni **Vefþjónustur** í [!INCLUDE[d365fin](includes/d365fin_md.md)], svo sem `https://mycompany.financials.dynamics.com:7048/MS/ODataV4/`.</span><span class="sxs-lookup"><span data-stu-id="becb9-112">For the *OData URL*, you can copy the OData V4 URL of any of the web services that are listed in the **Web Services** page in [!INCLUDE[d365fin](includes/d365fin_md.md)], such as `https://mycompany.financials.dynamics.com:7048/MS/ODataV4/`.</span></span>  

   <span data-ttu-id="becb9-113">Fyrir *Fyrirtækisheiti* skal nota heitið sem birtist í reitnum **Heiti** í glugganum **Stofngögn** í [!INCLUDE[d365fin](includes/d365fin_md.md)].</span><span class="sxs-lookup"><span data-stu-id="becb9-113">For the *Company Name*, use the name that is shown in the **Name** field in the **Company Information** window in [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span> <span data-ttu-id="becb9-114">Ef [!INCLUDE[d365fin](includes/d365fin_md.md)] notanda inniheldur mörg fyrirtæki skal velja viðeigandi heiti fyrirtækis af listanum í glugganum **Fyrirtæki**.</span><span class="sxs-lookup"><span data-stu-id="becb9-114">If your [!INCLUDE[d365fin](includes/d365fin_md.md)] contains multiple companies, choose the relevant company name from the list in the **Companies** window.</span></span> <span data-ttu-id="becb9-115">Í báðum tilvikum skal ganga úr skugga um að heiti sem tilgreind eru í leiðsagnarforriti fyrir Power BI samsvari nákvæmlega textanum í [!INCLUDE[d365fin](includes/d365fin_md.md)], svo sem `My Company`.</span><span class="sxs-lookup"><span data-stu-id="becb9-115">In both cases, make sure that the name that you specify in the Power BI wizard matches exactly the text shown in [!INCLUDE[d365fin](includes/d365fin_md.md)], such as `My Company`.</span></span>
3. <span data-ttu-id="becb9-116">Þegar búið er að færa inn upplýsingarnar er valinn hnappurinn Í lagi.</span><span class="sxs-lookup"><span data-stu-id="becb9-116">Once you have entered the information, choose the OK button.</span></span> <span data-ttu-id="becb9-117">Næsta skrefið í leiðsagnarforritinu verður að færa inn notandanafn og aðgangsorð.</span><span class="sxs-lookup"><span data-stu-id="becb9-117">The next step in the wizard will be to enter your username and password.</span></span>

   > [!NOTE]  
>    <span data-ttu-id="becb9-118">Ef aðrir sannvottunarkostir eru í boði á vinstra yfirlitssvæði skal velja *Grunnur*.</span><span class="sxs-lookup"><span data-stu-id="becb9-118">If there are other authentication options available in the left hand navigation, choose *Basic*.</span></span>
4. <span data-ttu-id="becb9-119">Slá inn notendanafn og aðgangsorð.</span><span class="sxs-lookup"><span data-stu-id="becb9-119">Enter your username and password.</span></span> <span data-ttu-id="becb9-120">Þessar upplýsingar má finna í glugganum **Notendur** í [!INCLUDE[d365fin](includes/d365fin_md.md)].</span><span class="sxs-lookup"><span data-stu-id="becb9-120">You can find this information in the **Users** window in [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span> <span data-ttu-id="becb9-121">Notið **Aðgangslykil fyrir vefþjónustu** sem aðgangsorðið þitt.</span><span class="sxs-lookup"><span data-stu-id="becb9-121">Use the **Web Access Key** as your password.</span></span>

   <span data-ttu-id="becb9-122">Til dæmis, notandanafnið þitt er *ADMIN*, og aðgangslykill vefþjónustu sem virkar sem aðgangsorð er *EgzeUFQ9Uv0o5O0lUMyqCzo1ueUW9yRF3SsLU=*.</span><span class="sxs-lookup"><span data-stu-id="becb9-122">For example, your username is *ADMIN*, and the web service access key that serves as your password is *EgzeUFQ9Uv0o5O0lUMyqCzo1ueUW9yRF3SsLU=*.</span></span>
5. <span data-ttu-id="becb9-123">Veldu hnappinn **Tenging** til að halda áfram.</span><span class="sxs-lookup"><span data-stu-id="becb9-123">Choose the **Connection** button to continue.</span></span> <span data-ttu-id="becb9-124">Leiðsagnarforritið fyrir Power BI birtir lista yfir [!INCLUDE[d365fin](includes/d365fin_md.md)]-gagnaveitur.</span><span class="sxs-lookup"><span data-stu-id="becb9-124">The Power BI wizard shows a list of [!INCLUDE[d365fin](includes/d365fin_md.md)] data sources.</span></span> <span data-ttu-id="becb9-125">Þessar gagnaveitur tákna allar vefþjónustur sem notandi hefur birt úr [!INCLUDE[d365fin](includes/d365fin_md.md)].</span><span class="sxs-lookup"><span data-stu-id="becb9-125">These data source represent all the web services that you have published from your [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span>

   <span data-ttu-id="becb9-126">Einnig er hægt að stofna nýja vefslóð vefþjónustu í [!INCLUDE[d365fin](includes/d365fin_md.md)] með því að nota aðgerðina **Stofna gagnamengi** á síðunni **Vefþjónustur** með því að nota Uppsetningu með hjálp fyrir **Setja upp skýrslugerð** eða með því að velja aðgerðina **Breyta í Excel** í hvaða lista sem er.</span><span class="sxs-lookup"><span data-stu-id="becb9-126">Alternatively, create a new web service URL in [!INCLUDE[d365fin](includes/d365fin_md.md)] by using the **Create Data Set** action in the **Web Services** page, using the **Set Up Reporting** Assisted Setup guide, or by choosing the **Edit in Excel** action in any lists.</span></span>

6. <span data-ttu-id="becb9-127">Tilgreinið gögnin sem notandi vill bæta við gagnalíkanið þitt og veljið svo hnappinn **Hlaða**.</span><span class="sxs-lookup"><span data-stu-id="becb9-127">Specify the data you want to add to your data model, and then choose the **Load** button.</span></span>
7. <span data-ttu-id="becb9-128">Endurtaktu fyrri skref til að bæta fleiri [!INCLUDE[d365fin](includes/d365fin_md.md)]-gögnum við Power Bi-gagnalíkanið þitt.</span><span class="sxs-lookup"><span data-stu-id="becb9-128">Repeat the previous steps to add additional [!INCLUDE[d365fin](includes/d365fin_md.md)] data to your Power BI data model.</span></span>

>    <span data-ttu-id="becb9-129">Þegar notanda hefur tekist að tengjast [!INCLUDE[d365fin](includes/d365fin_md.md)] verður hann ekki beðinn aftur um OData-vefslóð, notandanafn eða aðgangsorð.</span><span class="sxs-lookup"><span data-stu-id="becb9-129">Once you have successfully connected to [!INCLUDE[d365fin](includes/d365fin_md.md)], you will not be prompted again for the OData URL, username, or password.</span></span>
